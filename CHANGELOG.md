# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-13
### Added
- Initial implementation of PawnDex MCP Server.
- Tools: `search_packages`, `get_package`, `get_stats`, `get_languages`.
- FastMCP 3.x integration for efficient tool definition.
- HTTXP for asynchronous API calls to PawnDex.
- Support for local STDIO transport.
